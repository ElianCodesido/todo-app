# Task Manager App

A task management application built with React and TypeScript.

## Features

- Create multiple task lists
- Add, edit, and delete tasks
- Mark tasks as completed
- Draggable windows
- Local storage persistence

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
